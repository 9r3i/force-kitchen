# Usage
It's simple, just put this in kitchen section of ForceWebsite configuration file.
```json
{
  "kitchen": {
    "namespace": "kitchen",
    "host": "https://9r3i.github.io/force-kitchen",
    "key": "kitchen"
  }
}
```
